# Use Case Points – FitLife (Интернет-магазин спортивных товаров)

## Обозначения

**Unadjusted Actor Weight (UAW)** – «Вес акторов» оценивает сложность взаимодействия с системой.  
- **Простой** – взаимодействие через простой интерфейс (например, Email-сервис) – вес 1  
- **Средний** – взаимодействие через API, внешние сервисы, интеграции – вес 2  
- **Сложный** – взаимодействие через GUI с множеством действий или высокой сложностью – вес 3  

**Unadjusted Use Case Weight (UUCW)** – «Вес Use Case» оценивает сложность сценариев использования.  
- **Простой** – до 3 шагов, несложные действия – вес 5  
- **Средний** – 4–7 шагов, логическая обработка данных – вес 10  
- **Сложный** – больше 7 шагов, сложная логика, интеграции – вес 15  

**Technical Complexity Factor (TCF)** – коэффициент технической сложности.  
**Environmental Complexity Factor (ECF)** – коэффициент сложности среды (команды, опыта и др.)

## 5. Расчёт Use Case Points (UCP)

**Формула:**  
`UCP = (UAW + UUCW) × TCF × ECF`  

`UCP = (24 + 100) × 1.05 × 0.845 ≈ 89 UCP`  

**Итоговая оценка проекта FitLife:**  
**Unadjusted Actor Weight (UAW) = 24**  
**Unadjusted Use Case Weight (UUCW) = 100**  
**Technical Complexity Factor (TCF) = 1.05**  
**Environmental Complexity Factor (ECF) = 0.845**  
**Use Case Points (UCP) ≈ 89**
