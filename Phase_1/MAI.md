# ПМ  
**Фаза 1**

## Часть 1: Концепции программного проекта

1. **Бот для изучения языков (Language Bot)**  
   ○ Описание: Telegram/Discord бот для изучения иностранных языков. Предоставляет ежедневные уроки, словарные тесты, отслеживает прогресс пользователя, предлагает разговорные практики и интеграцию с API для перевода и аудио. Пользователи могут выбирать уровень сложности и языки, получать напоминания.

2. **Менеджер рецептов (Recipe Manager)**  
   ○ Описание: Веб-приложение или мобильное приложение для управления рецептами. Пользователи добавляют свои рецепты, ищут по ингредиентам, планируют меню на неделю, рассчитывают калорийность. Интегрируется с API питания (например, Nutritionix) для анализа ингредиентов и рекомендаций.

3. **Трекер привычек (Habit Tracker)**  
   ○ Описание: Мобильное приложение для формирования привычек. Пользователь ставит цели, отмечает выполнения, приложение показывает статистику, графики, серии успехов, напоминания и мотивационные элементы.

4. **Интернет-магазин спортивных товаров (FitLife)**  
   ○ Описание: Веб-приложение (интернет-магазин) для спортивного клуба "FitLife". Пользователи просматривают каталог товаров (одежда, инвентарь, аксессуары), описания, фото, добавляют в корзину, оформляют заказы с интеграцией оплаты (ЮKassa, СБП) и доставки (СДЭК, Boxberry). Отзывы, рейтинги, поиск, фильтры. Админ-панель для управления товарами, заказами, пользователями. Уникальный дизайн, быстрая загрузка, удобный UX для высокой конкуренции.

## Часть 2: Критерии оценки

- Сложность реализации (Implementation Complexity)  
- Потенциал для демонстрации (Demonstration Potential)  
- Практическая полезность (Practical Usefulness)  
- Охват всех фаз (охват тем ПМ)

## Анализ результатов

- **Recipe Manager — лидер (0.4899, ранг 1)**: Высокие баллы по всем критериям, особенно по практической полезности и демонстрации. Низкая сложность, повседневная ценность (меню, калории), хорошая интеграция API.

- **Language Bot — второе место (0.2066, ранг 2)**: Хорошая демонстрация (интерактивный бот), средняя сложность, полезность для образования. Минус — сезонность или нишевость.

- **FitLife — третье место (0.1151, ранг 3)**: Наш проект. Высокая сложность (e-commerce с интеграциями), хорошая демонстрация (живой сайт, заказы), практическая полезность (коммерция для клуба). Минус — большой охват тем ПМ, но риски в реализации (бюджет, сроки).

- **Habit Tracker — последнее (0.0748, ранг 4)**: Высокая полезность (личное развитие), но средняя сложность и демонстрация (графики, напоминания), узкий охват.

FitLife на 3 месте — средний результат. Он сложный, но полезный для бизнеса. Для улучшения ранга — упростить MVP или подчеркнуть уникальность (для клуба FitLife).
