# Фаза 2. Планирование и распределение работ проекта

В ходе **Фазы 2** были выполнены работы по планированию проекта создания интернет-магазина спортивных товаров для спортивного клуба **FitLife**.

---

## 1. Декомпозиция работ

Проект был декомпозирован на следующие группы работ:

- анализ и формализация требований;
- проектирование архитектуры интернет-магазина;
- разработка backend-части;
- разработка frontend-части;
- интеграция платёжных и логистических сервисов;
- тестирование и подготовка к запуску.

Каждая группа была разбита на конкретные задачи, пригодные для оценки и распределения между участниками команды.

---

## 2. Оценка усилий

Оценка трудоёмкости задач проводилась с использованием метода **Planning Poker**.  
Оценка осуществлялась двумя участниками команды с учётом:

- технической сложности задач;
- рисков интеграции сторонних сервисов (оплата, доставка);
- ограничений по срокам и бюджету проекта.

Результаты оценки были зафиксированы в электронной таблице и использовались для балансировки нагрузки между участниками проекта.

## 4. Выводы по Фазе 2

В рамках **Фазы 2** была выполнена декомпозиция проекта и произведена экспертная оценка трудоёмкости работ.
